## Applied Machine-Learning Mini-Projects
<p>Individual Mini-projects I did for CS208 - Machine Learning for Data Science at Fulbright University Vietnam. <p>

## Mini-Project List
My 5 class projects, each working on a dataset with distinct sets of machine learning algorithms, includes:
- Travel Insurance Purchase Prediction with Gaussian/Mixed Naive Bayes.
- Clustering User by Credit Card Use Behavior with K-means and Principle Component Analysis.
- Job Change Prediction with Decision Tree/Random Forest.
- Breast Cancer Prediction with Support Vector Machine and Model Ensembles (Boosting, Including AdaBoost and XGBoost, Stacking, Bagging).
- Simple Image Classification (CIFAR10) with Deep Neural Network.

<p>Look at main notebooks in each mini-project for more information.<p>

## Environment Set-up and Running
<p> The conda/mamba package and environment manager is used. Create the environment with the following command: <p>

```bash
mamba create -f environment.yml
```
Then run the `.ipynb` notebook files with the kernel corresponding to the environment.
